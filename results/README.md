# Результаты

Здесь хранится эталонный вывод конвейера на трёх пилотных контурах
сельского поселения Манжерок (см. `data/boundary_*.geojson`).

## `results.json`

Единый JSON-файл с полной диагностикой прогона. Структура:


## Воспроизводимость

`results.json` соответствует прогону от 2026-04-13 на полной выборке
196 695 участков. Любой повторный запуск `pipeline_v84.py` на тех же
входных данных должен давать идентичные значения CV-метрик с точностью
до случайности подвыборки 3000 контрольных точек (фиксируется зерном
внутри конвейера).

## Что ещё генерирует конвейер

При запуске `pipeline_v84.py` создаётся папка `results_YYYY-MM-DD_HH-MM-SS/`
с дополнительными артефактами:

- `results.json` — этот файл,
- `01_prior.svg` — приор по классам,
- `02_alpha.svg` — калибровочная кривая α,
- `03_kde_*.svg` — KDE по непрерывным признакам в разрезе ВРИ,
- `04a_cv_accuracy.svg`, `04b_cv_ranks.svg` — результаты CV,
- `05_pe_matrix.svg` — матрица P × E для пилотов,
- `06_boundaries_map.png` — картограмма пилотных контуров.

В репозитории хранится только `results.json` как самый компактный
и информативный артефакт; SVG-визуализации воспроизводятся локально.
