# Данные

В этой папке лежат малые входные файлы, которые версионируются вместе
с кодом. Большие исходные слои (полная выборка ЕГРН и ЗОУИТ Республики
Алтай) опубликованы в
[GitHub Releases](https://github.com/nikert/vri-recommender/releases/tag/data-v1).

---

## Файлы в репозитории

### `classifier_vri.csv`
Классификатор видов разрешённого использования по Приказу
Росреестра от 10.11.2020 № П/0412 (с учётом изменений). 171 позиция
с трёх- и четырёхуровневой иерархией (1.0 → 1.1 → 1.2…).

| Колонка | Описание |
| :--- | :--- |
| Код | Числовое обозначение ВРИ (например, `1.0`, `2.5.1`) |
| Наименование | Краткое наименование ВРИ |
| Описание | Текст из приказа Росреестра |

Кодировка: UTF-8, разделитель: `;`, кавычки: двойные.

В пилотном расчёте используются только укрупнённые категории первого
уровня (1.0 … 13.0); код 14.0 исключён, поскольку соответствует нулевому
функциональному использованию участка.

### `boundary_1.geojson` / `boundary_2.geojson` / `boundary_3.geojson`
Три пилотных контура в сельском поселении Манжерок, Майминский район,
Республика Алтай, для проверки рекомендательной системы на новых
территориях:

| Файл | Площадь | Уклон (среднее) | POI density | Расстояние до УДС |
| :--- | ---: | ---: | ---: | ---: |
| `boundary_1.geojson` | 81,22 га | 0,93° | 5,35 пт/км² | 1123 м |
| `boundary_2.geojson` | 34,59 га | 13,02° | 0,05 пт/км² | 2997 м |
| `boundary_3.geojson` | 2,94 га | 3,16° | 6,06 пт/км² | 389 м |

CRS: EPSG:4326 (WGS84). Геометрия — `MultiPolygon`. Атрибуты не
требуются: пилотные контуры обрабатываются как чисто геометрические
маски.

### Скачать вручную

Перейти по ссылке релиза, скачать файлы из секции «Assets»:
<https://github.com/nikert/vri-recommender/releases/tag/data-v1>

### Скачать через `curl`

```bash
mkdir -p data_full && cd data_full

curl -L -o egrn.gpkg \
  https://github.com/nikert/vri-recommender/releases/download/data-v1/egrn_features.gpkg

curl -L -o zouit.geojson \
  https://github.com/nikert/vri-recommender/releases/download/data-v1/zouit.geojson
```

### Проверка целостности

```bash
sha256sum egrn.gpkg zouit.geojson
```

Контрольные суммы должны совпадать со значениями из таблицы выше.
Расхождение — повод повторить скачивание.

## Источники

- **ЕГРН (Единый государственный реестр недвижимости).** Открытые
  фрагменты по Республике Алтай. Первоисточник — Росреестр.
- **ЗОУИТ (Зоны с особыми условиями использования территорий).**
  Сводный слой из открытых данных Росреестра по Республике Алтай,
  агрегированы по 10 типам охранных зон (Z01…Z10).
- **POI и УДС.** OpenStreetMap, выгрузка через Overpass API; в публичный
  релиз не включены — извлекаются на стороне пользователя при необходимости.
- **DEM.** Свободные источники (SRTM 1 arc-second / Copernicus
  GLO-30); в релизе отсутствует.
